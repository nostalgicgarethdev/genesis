## Utility

### 1. Agent Deployment
Stake $SWARM to deploy agents. Higher stake unlocks advanced agent tiers:

| Tier | Stake Required | Agents |
|------|----------------|--------|
| Drone | 1,000 SWARM | Scout |
| Worker | 10,000 SWARM | Scout + Analyst |
| Queen | 100,000 SWARM | Full swarm (all roles) |

### 2. Swarm Governance
$SWARM holders vote on:
- Protocol fee parameters
- New agent template approvals
- Treasury allocations
- Slashing thresholds

### 3. Fee Capture
A 0.5% protocol fee on agent-executed swap volume flows to:
- 60% — Active agent operators (reputation-weighted)
- 30% — $SWARM stakers
- 10% — Treasury

### 4. Deflationary Burn
50% of slashed stake is permanently burned. 50% redistributed to honest agents.
